User Network Clusters

Network Visualization

This network visualization shows how users are connected based on their tweet content similarity. Nodes represent users, and edges represent content similarity between users.

Cluster Visualization

This visualization shows how users are grouped into clusters based on their tweet content. Users in the same cluster tend to have similar tweeting patterns.

About These Visualizations

These visualizations provide two different views of how users are related:

  • Network View: Shows direct connections between users based on content similarity
  • Cluster View: Shows how users are grouped into distinct clusters

How to interpret:

  • Nodes represent users
  • Edges represent content similarity
  • Colors represent different clusters
  • Closer nodes are more similar
