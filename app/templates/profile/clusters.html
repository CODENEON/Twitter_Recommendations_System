About User Clustering

This visualization shows how users are grouped based on their tweet content, sentiment patterns, and hashtag usage. Each cluster represents a group of users with similar behavior patterns.

How to interpret:

  • Users in the same cluster tend to have similar tweeting patterns
  • The distance between points indicates similarity
  • Clusters are formed using K-means clustering on user features
